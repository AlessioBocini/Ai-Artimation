Ai Artimation

Estimation of Art similarities with Ai

A website where the pinnacle of technology is put at the center, where art and technology go hand in hand.
The website has been provided with a collection of paintings from which the algorithm of artificial intelligence can select various pairs of works and if they are similar, these will be displayed in a list and from this you can choose and access each of these to display their description and what else.
Under each pair of paintings there is a set of keywords related to that specific pair, and if you are curious and you are looking for a particular matching, you just need to type in the search bar a certain word or the name of an artist, all the pairs of paintings containing that specific keyword will be shown below.

More Details

The website was fully built in html5 and css markup languages.
When the website is opened, the algorithm of artificial intelligence will collect each work from the collection and it’ll calculate the similarity of each possible pair of works.
The algorithm of Artificial intelligence we used was the Universal Sentence Encoder, which calculates the similarities of two or multiple texts depending on the repetition of words.
Each painting is equipped with both a textual description and a set of tags, both will be the subjects of the calculation of the similarity.
Two works may be similar either in terms of textual description or in terms of similar tags, so, the algorithm will calculate for each pair two distinct values of similarity.
To simplify our objective, we will use a global value of similarity obtained by doing a weighted average of the two individual values of similarity.
The decision to calculate two distinct values of similarity, to start with, comes from two critical points regarding this algorithm, It’s not capable to take two different phrases with an equal meaning and tell they are equal yet and if two texts contain multiple punctuation marks and there is a wide use of conjunctions, their presence will affect positively the grade of similarity between texts despite these are not similar at all.
An advantage in using this global value of similarity is that it’s possible to use a much higher threshold of acceptance.
If a pair of works has a global value of similarity which crosses that said threshold, it’s classified as probable and so will be catalogued in the list of pairs shown in the website, and so explorable by the user.
Each pair of paintings will have a set with at least one keyword, these will be the tags the paintings have in common.
The existence of tags, becomes even more important when we begin to use the research system.
When we search for a certain word, the list of displayed pairs will be reduced displaying only pairs which contain that specific word.
